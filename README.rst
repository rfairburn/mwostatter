========
MWOStatter
========

`MWOStatter` is a Python parser for your MWO stats.

Prerequisites
=============
pyyaml
requests

Usage
=====

Place a config file as follows in the same folder as mwostatter.py named mwostatter.ini

.. code-block::
  
    [Mechwarrior]
    email = <your_mwo@email.address>
    password = <your_mwo_password>
